# 架构设计文档

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│              GitHub Trending Digest                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌──────────────┐    ┌──────────────────────────────────┐  │
│  │   CLI 层    │    │         Scheduler 层               │  │
│  │  cli/main.py │    │     定时任务 (schedule)            │  │
│  └──────┬───────┘    └──────────────┬───────────────────┘  │
│         │                            │                       │
│         └────────────────────────────┼───────────────────────┘
│                                      │                       │
│  ┌───────────────────────────────────┴────────────────────┐ │
│  │                    核心处理层                           │ │
│  │  ┌────────────┐  ┌────────────┐  ┌────────────┐       │ │
│  │  │  Fetcher   │  │  Analyzer  │  │  Storage   │       │ │
│  │  │  数据抓取  │  │  数据分析  │  │  数据存储  │       │ │
│  │  └────────────┘  └────────────┘  └────────────┘       │ │
│  └──────────────────────────────────────────────────────────┘ │
│                              │                                │
│  ┌───────────────────────────┴─────────────────────────────┐ │
│  │                    通知层                              │ │
│  │      WeChatNotifier      │      ConsoleNotifier        │ │
│  └──────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────┘
```

## 模块职责

### 1. Fetcher (fetcher.py)
- 抓取 GitHub Trending 页面
- 调用 GitHub API 获取项目信息
- 支持多语言筛选

### 2. Analyzer (analyzer.py)
- 数据清洗和排序
- AI 项目智能筛选
- 语言/分类统计

### 3. Storage (storage.py)
- SQLite 本地存储
- 历史数据查询
- 报告持久化

### 4. Notifier (notifier.py)
- WeChat 推送（通过 QClaw Gateway）
- Console 输出（调试用）

### 5. Scheduler (scheduler.py)
- 每日任务调度
- 每周任务调度
