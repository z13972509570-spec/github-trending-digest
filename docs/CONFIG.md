# 配置说明

## 环境变量

| 变量 | 说明 | 必需 |
|------|------|------|
| `GITHUB_TOKEN` | GitHub PAT | 推荐 |
| `QCLAW_GATEWAY_URL` | QClaw Gateway 地址 | 微信推送时 |
| `QCLAW_TOKEN` | QClaw Token | 微信推送时 |
| `WECHAT_TARGET` | 微信目标用户 | 微信推送时 |
| `DAILY_PUSH_TIME` | 每日推送时间 | 可选 |
| `WEEKLY_PUSH_DAY` | 周报推送日 | 可选 |

## 数据库

数据存储在 `./data/trending.db`
